Atividade PHP Orientado a Objetos

Atividade 1

Crie uma função que receba 2 números e retorne um array associativo contendo a média e também um indicador booleano de aprovado/reprovado.
Considere aprovado com média >= 6.

Atividade 2

Crie uma função que receba um array associativo contendo nota e peso, realize a média das notas considerando o peso. Exemplos:
Lista com 2 notas: (N1*P1) + (N2*P2) / 2 = Resultado
Lista com 3 notas: (N1*P1) + (N2*P2) + (N3*P3) / 3 = Resultado

Atividade 3

Crie um programa para cadastrar, listar e excluir produtos de uma lista contendo nome e preço.

*O código original da atividade está comentado no codígo desta atividade no github.
